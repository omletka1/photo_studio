О проекте

**Photo Studio** — это современная веб-платформа для организации и проведения фотоконкурсов. Система позволяет фотографам-любителям и профессионалам участвовать в конкурсах, получать обратную связь от жюри и зрителей, а также демонстрировать свои работы широкой аудитории

| Функция | Описание |
|---------|----------|
| Управление конкурсами | Создание конкурсов с настраиваемыми сроками, описанием и категориями номинаций |
| Загрузка работ | Участники могут добавлять до 5 изображений на одну работу с подробным текстовым описанием |
| Оценка жюри | Члены жюри выставляют баллы по 5-балльной шкале и оставляют структурированные комментарии |
| Зрительское голосование | Зарегистрированные пользователи могут голосовать за понравившиеся работы (один голос на работу) |
| Система обсуждения | Древовидные комментарии, позволяющие жюри и администраторам общаться |
| Панель администратора | Централизованный интерфейс для управления пользователями, настройки конкурсов, просмотра статистики |
| Безопасная авторизация | Регистрация с подтверждением email, маршрутизация и контроль доступа на основе ролей |
| Адаптивный интерфейс | Современный UI на базе Tailwind CSS, включающий ленивую загрузку, галерею Lightbox и плавные анимации |

Платформа решает следующие задачи:
- Централизованная публикация и хранение фоторабот
- Прозрачная и структурированная система оценки членами жюри
- Интерактивный механизм голосования для вовлечения аудитории
- Ролевая модель доступа для администраторов, жюри и участников
- Современный адаптивный интерфейс, оптимизированный для настольных и мобильных устройств
